Hello, Gateway

API

  • Get data from sensor

    [GET]http://{{ip}}:{{port}}/api/data?name= {sensorid1}& name= {sensorid2}
    [GET]http://{{ip}}:{{port}}/api/{user}/data?name= {sensorid1}& name= {sensorid2}
    Example:
    http://{{ip}}:{{port}}/api/data?name=light&name=thermometer1
    http://{{ip}}:{{port}}/api/david@cisco.com/data?name=light&name=thermometer1
    Response:
    { "thermometer1": { "temperature": 25.95 }, "light": { "brightness": 15 } }

  • Activate sensor action

    [PUT,POST]http://{{ip}}:{{port}}/api/action?name={sensorId}&action={actionName}&{actionParamater}=value
    [PUT,POST]http://{{ip}}:{{port}}/api/{user}/action?name={sensorId}&action={actionName}&{actionParamater}=value
    Example:
    http://{{ip}}:{{port}}/api/action?name=led_11&action=flash&duration=12&interval=1
    http://{{ip}}:{{port}}/api/david@cisco.com/action?name=led_11&action=flash&duration=12&interval=1

    • Modify the sensor setting

    [PUT,POST]http://{{ip}}:{{port}}/api/modify
    [PUT,POST]http://{{ip}}:{{port}}/api/{user}/modify
    Example:
    http://{{ip}}:{{port}}/api/modify
    http://{{ip}}:{{port}}/api/david@cisco.com/modify
    Payload:
    { "id":"PersonalDevice", "settings":[ { "name":"mac_address", "value":"1234567" } ] }
